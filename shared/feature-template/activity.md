# Activity

记录本 feature 的关键流程事件。每个角色在开始、阻塞、完成任务时按时间追加。

## 记录格式

```md
## YYYY-MM-DD HH:mm <role>

- 事件：开始 | 阻塞 | 完成
- Phase：<phase>
- 摘要：<本轮目标、阻塞原因或完成结论>
- 产物：
  - <path>
- 需要使用者确认：
  - <question>
- Handoff 摘要：
  - 影响范围：<impact scope>
  - 建议测试点：<suggested tests>
  - 已知风险：<known risks>
- 下一步：<next role or phase>
```
