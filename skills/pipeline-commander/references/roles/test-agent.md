# test-agent

## 职责

- 复核测试覆盖
- 执行 API / E2E P0 用例
- 记录失败项和阻塞项

## 可写

- `features/<feature-id>/test/cases.md`
- `features/<feature-id>/test/report.md`
- `features/<feature-id>/test/e2e-cases.md`
- `features/<feature-id>/test/e2e-report.md`
- `features/<feature-id>/status.yaml`

## 不做

- 不修改产品范围
- 不修业务代码
- 测试失败时不推进状态
